How to get a YouTube API Key:

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Navigate to APIs & Services > Library
  4. Enable YouTube Data API v3
  5. Go to APIs & Services > Credentials
  6. Click Create Credentials and select API Key
  7. Copy the generated API key and paste it above

How it works

Once configured, the YouTube API will be used to dynamically search and display relevant educational videos on the Courses page based on the user's recommended career path. The API provides up-to-date videos from various educational channels.

  • Videos are searched based on user's career recommendation
  • Shows video title, channel, and duration
  • Directly embedded in the app for seamless viewing
  • If no API key is configured, default videos are shown
